# Food Court Management

A React + Vite starter app for food court and catering management.

## Features

- Food court order/token dashboard
- Daily sales and catering pipeline summary
- Catering booking list with guest count and quotation value
- Menu management with search
- Inventory reorder alerts
- Responsive UI for desktop and mobile

## Live preview while coding

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Vite will show a local preview URL, usually:

```bash
http://localhost:5173
```

When you edit files in `src/`, the browser updates automatically with hot reload.

## Production build

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Main files

- `src/App.jsx` — main dashboard UI
- `src/styles.css` — Tailwind and global styles
- `tailwind.config.js` — Tailwind configuration
- `package.json` — scripts and dependencies
